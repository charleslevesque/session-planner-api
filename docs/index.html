SessionPlanner API

A RESTful .NET API for managing laboratory sessions, teaching needs, infrastructure resources, and personnel in an educational environment. Built with a layered architecture: Controllers → Services → DbContext. Integrated with OpenAI for intelligent form assistance, rejection guidance, and administrative review.
